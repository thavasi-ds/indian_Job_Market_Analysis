 📊 Indian Tech Job Market Analysis - EDA

📌 Project Overview
    
     •This project performs Exploratory Data Analysis (EDA) on the Indian tech job market to identify hiring trends, salary patterns, and in-demand skills.

 🎯 Business Objectives

     Analyze Indian tech job market trends
     •Identify cities offering highest salaries
     •Find skills with high salary premiums
     •Compare Remote vs Hybrid vs On-site opportunities
     •Understand fresher opportunities across cities
     •Identify companies hiring for ML roles


🔍 Analysis Performed

     •Data Cleaning
     •Exploratory Data Analysis
     •Salary distribution analysis


 📊 Visualizations


      •Company hiring trends

🚀 Conclusion

  This analysis provides insights into the Indian technology job market by identifying salary trends, demanded skills, and hiring patterns. These findings can help freshers and professionals make informed career decisions.
